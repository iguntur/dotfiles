# My NVIM

## Keymaps

Mode:

```
- normal (n)
- insert (i)
- visual (v)
```

Key:

```
- C: Control
- M: Alt (windows/linux) / Option (mac)
- S: Shift
- <Leader>: comma (,)
- k: Up
- j: Down
```

> `(mod) key(arguments...) : description`

-   `(n) M-k : move line go up`
-   `(n) M-j : move line go down`
